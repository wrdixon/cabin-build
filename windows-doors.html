Windows & Doors In Progress

Marvin Elevate windows — Therma-Tru entry door — door RO is current blocker for all exterior work

Overview

Eight Marvin Elevate windows and one Therma-Tru Smooth-Star entry door. Window rough openings are correctly framed and verified. The door rough opening is currently too narrow and must be enlarged before any exterior work proceeds — this is a hard blocker.

Decisions

1. Window product

Marvin Elevate series throughout, per order schedule.

Reason: good thermal performance for Zone 6; consistent product line simplifies ordering and warranty.

2. Door product

Therma-Tru Smooth-Star, 3/0×6/8, left-hand inswing, prehung.

Reason: fiberglass door performs well in Vermont climate — won’t warp or rot, and provides good thermal performance.

3. Window layout

Per window schedule (verify all sizes and quantities against Marvin order confirmation before framing is closed):

  • South face: 3 picture windows (~54×54"), 1 entry door
  • East gable end: 2 double-hung windows (~48×48")
  • West gable end: 2 double-hung windows (~48×48")
  • North face: 1 picture window (~54×54")
  • Loft: 1 window (size TBD)

Reason: south glazing maximizes passive solar gain. Gable-end double-hungs allow cross ventilation. North window provides light without significant heat loss.
