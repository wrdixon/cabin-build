Chimney Open

Through-roof class A chimney system for wood stove — stove collar to cap

Overview

The wood stove chimney will exit through the roof. The route passes through the interior ceiling space, the roof assembly including polyiso and strapping, and terminates above the roof with a cap. The chimney penetration must be located and constructed before the roof assembly is finalized. This page covers the full chimney system from stove collar to cap.

Decisions

1. Chimney routing

Through the roof rather than through a wall.

Reason: through-roof is more efficient (shorter run, better draft), avoids a wall penetration in the insulated envelope, and is standard for freestanding stoves. Wall exit would require a horizontal run and tee which adds complexity and reduces draft.

Installation Notes

  1. Finalize stove location before any chimney work. All chimney dimensions flow from stove placement.
  2. Locate chimney penetration on roof before finalizing strapping layout and panel cuts. This must happen before roofing is complete.
  3. Frame penetration opening with appropriate clearance to combustibles per class A pipe spec.
  4. Install class A chimney pipe from stove collar through ceiling and roof. Follow manufacturer clearance requirements at every framing intersection.
  5. Install roof flashing integrated with standing seam panels. Coordinate with roofer on flashing product and method.
  6. Install outside air kit: sleeve and flash 4″ penetration through wall assembly before siding goes on. Run 4″ flex conduit from exterior cap to stove collar area. Install rodent screen at exterior cap.
  7. Install chimney cap at top. Confirm height meets 2 ft / 10 ft rule.

Open Items

  • Stove location: not yet finalized. Chimney location depends on stove placement. Stove must be located before chimney is planned. Consider clearances to combustibles, outside air kit routing, and visual placement in the space. See wood-stove.html.
  • Chimney system product: insulated double or triple wall class A chimney pipe required for through-roof installation. Brand and diameter TBD — must match Jøtul F 602 V3 6″ flue collar.
  • Roof penetration detail: chimney must pass through strapping, polyiso, and roofing layers. Flashing at roof surface must be integrated with standing seam panels. This is a sequencing blocker — chimney location must be set before roof panels are laid out and cut. See roof-assembly.html.
  • Chase or open pipe: does the chimney need a chase enclosure for aesthetics or code? Confirm with Dummerston.
  • Height above roof: code requires minimum 2 ft above any point within 10 ft horizontally. Confirm final height based on roof pitch and penetration location.
  • Clearances to combustibles: confirm required clearances from chimney to framing at ceiling and roof penetration. Class A pipe has reduced clearances vs. single wall — confirm spec for the selected pipe product.
  • Outside air kit routing: 4″ flex metal conduit from exterior wall to stove. Penetration through wall assembly must be sleeved and flashed before siding goes on. See siding.html.
  • Hire chimney installer or self-install: confirm with Forest.
